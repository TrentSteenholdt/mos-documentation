    Admin: Exam Management

    Access Level

    Owner (Administrator)

    Overview

    Build and administer exams using reusable question banks. Assign to users or subgroups with time windows and completion requirements. Analyse results and export for records.

    Create an Exam

    1. Click Create New Exam.
    2. Enter title, description/instructions.
    3. Add questions (from bank or new) and set order.
    4. Save as Draft.

    Question Types

    The system supports three question types:

    • Single: Multiple choice with one correct answer.
    • Multiple: Multiple select where users select all correct options.
    • True/False: Binary true or false question.

    Creating Questions

    Add questions to build your exam:

    1. Open the exam and click Add Question.
    2. Choose a question type: Single, Multiple, or True/False.
    3. Enter the question text.
    4. Add answer options and mark the correct answer(s).
    5. Click Save Question.

    Tips:

    • Use clear wording and avoid ambiguity.
    • Mix difficulty levels and question types.
    • For Multiple type questions, select all correct answers.

    Scoring and Settings

    • Points per question and overall weighting.
    • Pass mark as percentage or raw score.
    • Attempts allowed (1, limited, or unlimited within window).
    • Time window: Start/end dates; Time limit per attempt (optional).
    • Must-complete flag: Blocks other workflows until completed (use sparingly).
    • Accommodations: Extended time for specific users (if supported).

    Publish and Assign

    1. Set exam to Published when ready.
    2. Assign to individual users, subgroups, or entire group.
    3. Configure availability window and reminders.
    4. Notify assignees by email (if enabled).

    Monitor and Results

    • Attempts: Started vs. completed counts.
    • Scores: Individual and average scores; pass/fail.
    • Question analysis: Difficulty index, most missed items (where available).
    • Export: Download results to CSV for archiving/audits.

    Troubleshooting

    • Users can’t see exam: Check publish state, assignment scope, and date window.
    • Results missing: Ensure attempts were submitted; check filters.

    Tips

    • ✅ Pilot exams with a small group first.
    • ✅ Mix question types to test understanding, not memorisation.
    • ✅ Keep instructions clear and include expected duration.
