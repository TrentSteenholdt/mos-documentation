    Admin: Depth Charting Overview

    Access Level

    Selector can view rankings and coverage, set manual rank order, and enter subjective scores. Owner can also run full recalculation.

    Overview

    Use Overview to manage the cross-grade ranking list for a season.

    This page shows:

    • Overall rank order across all grades
    • Each official's best grade and depth score
    • Grade coverage summary
    • Manual rank movement (for selectors)

    How to Access

    From the sidebar, expand Depth Charting, then click Overview.

    Before You Start

    • Select a season from the Season list before using the table.
    • If no rankings appear, run a recalculation first (Owner access required).
    • Subjective scoring buttons appear only when your group has active subjective metrics.

    Page Sections

    Grade Coverage

    The Grade Coverage panel shows one chip per grade with tier counts.

    • Click a grade chip to open that grade in Grade Chart
    • Border colours show coverage health at a glance

    Overall Ranking

    The Overall Ranking table shows:

    • Rank
    • Official
    • Best Grade
    • Depth Score
    • Subgroups
    • Subjective Scoring (selectors only, when metrics exist)

    Search Official

    Use Search official... in the page header to find an official in the current season table.

    • Type at least 2 characters
    • Click a result to jump to and highlight that row

    Fields and Inputs

    Season, required

    • Chooses which season rankings and coverage are loaded

    Bulk Score: Metric, required

    • Selects which subjective metric score to apply

    Bulk Score: Score (0-100), required

    • Must be between 0 and 100

    Bulk Score: Notes, optional

    • Adds optional notes with the score

    Bulk Score: Apply to, required

    • All grades applies to each selected official in all grades they are ranked in
    • Best grade only applies only to each selected official's best grade

    Common Tasks

    Load Overview for a Season

    1. Open Depth Charting > Overview.
    2. Select a value in Season.
    3. Wait for the ranking table and coverage chips to load.
    4. Result: You see the season's overall ranking and grade coverage.

    Enter Subjective Scores for One Official (Selector)

    1. In the official row, click Edit in Subjective Scoring.
    2. Enter score values in Score (0-100).
    3. Add optional notes.
    4. Click Save Scores.
    5. Result: Scores are saved for that official and a confirmation appears.

    Bulk Apply Subjective Score (Selector)

    1. Click Bulk Score.
    2. Select Metric.
    3. Enter Score (0-100).
    4. Choose All grades or Best grade only.
    5. Tick one or more officials.
    6. Click Apply Score.
    7. Result: The score is applied to selected officials.

    Reorder Overall Ranking (Selector)

    1. Drag rows using the handle at the left of each row.
    2. Drop each row in your preferred order.
    3. Result: Order saves automatically and a success message appears.

    Reset Manual Overall Ordering (Selector)

    1. Click Reset Order.
    2. Confirm the prompt.
    3. Result: Manual order is removed and ranking returns to automatic score order.

    Export Overall Ranking

    1. Select a season.
    2. Click Export CSV.
    3. Result: The season's overall ranking downloads as a CSV file.

    Recalculate All Grades (Owner)

    1. Click Recalculate.
    2. Wait for progress to finish for each grade.
    3. Result: Rankings are rebuilt and the page refreshes with new results.

    Validation and Rules

    • Search requires at least 2 characters
    • Bulk score must be between 0 and 100
    • At least one official must be selected for bulk score
    • Manual ordering is available to selectors only

    Troubleshooting

    No depth chart data for this season yet

    Symptoms
    The page shows no ranking rows for the selected season.

    How to Fix

    1. Confirm the correct Season is selected.
    2. If you have Owner access, click Recalculate.
    3. Reload the page after recalculation completes.

    Common Causes

    • Rankings have not been calculated yet for that season
    • The selected season has no eligible data yet

    Bulk score will not save

    Symptoms
    You see a warning when applying a bulk score.

    How to Fix

    1. Enter a score from 0 to 100.
    2. Tick at least one official.
    3. Confirm a metric is selected.

    Common Causes

    • Score is blank or outside range
    • No officials were selected
