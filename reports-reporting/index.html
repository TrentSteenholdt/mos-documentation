    Reporting

    Access Level

    All users (Users, Viewer, Reviewer, Owner) - some reports are restricted to specific roles

    Overview

    Filters and Export

    Control the time window and export the current view.

    Date Range

    1. Tick Specify a date range.
    2. Enter Start and End dates.
    3. Click Refresh to update charts and tables.

    Export to CSV

    1. Click Export this data to CSV.
    2. A CSV file downloads with the current report data.

    Tips:

    • Use smaller date ranges to focus on recent activity.
    • Refresh whenever you change date inputs.

    Tabs and Reports

    Navigate the tabs to explore different insights.

    Activity

    • Shows last logins and overall activity trends.
    • Optional: Breakdown by season/year to see year-by-year trends.

    How to use:

    1. Open Activity.
    2. Toggle Breakdown by season/year if needed.
    3. Review the Users Table and Activity Charts.

    Per Template

    • Focus on a specific assessment template.
    • Choose Diary, Match Day, or Peer, then select a Template.
    • See Template Details Data and Template Ratings Data.

    How to use:

    1. Open Per Template.
    2. Select Diary, Match Day, or Peer.
    3. Choose a Template from the dropdown.
    4. Review charts for details and ratings.

    Assessments

    • Summary of assessments and plans across their lifecycle.
    • Affected by the selected date range.

    How to use:

    1. Open Assessments.
    2. Adjust the date range if required.
    3. Review summary charts.

    Assessment Breakdown

    • Detailed breakdown of totals, types, and reviewed diary entries.
    • Optional: Breakdown by season/year.

    How to use:

    1. Open Assessment Breakdown.
    2. Toggle Breakdown by season/year.
    3. Review detailed charts.

    Exams

    • Tracks exam attempts, scores, and pass rates.
    • Optional: Breakdown by season/year.

    How to use:

    1. Open Exams.
    2. Toggle Breakdown by season/year.
    3. Review exam charts.

    Observers

    • Analyze observer feedback and engagement.
    • Optional: Breakdown by season/year.

    How to use:

    1. Open Observers.
    2. Toggle Breakdown by season/year.
    3. Review observer charts and notes.

    Common Tasks

    Export Recent Activity

    1. Click Reporting.
    2. Tick Specify a date range and set the last 30 days.
    3. Click Refresh.
    4. Click Export this data to CSV.

    Review a Specific Template’s Performance

    1. Click Reporting.
    2. Open Per Template.
    3. Select Diary or Match Day.
    4. Choose the relevant Template.
    5. Review Details and Ratings charts.

    Check Exam Outcomes for the Season

    1. Click Reporting.
    2. Open Exams.
    3. Tick Breakdown by season/year.
    4. Review trends and pass rates.

    Troubleshooting

    No data displayed

    • Try expanding the date range.
    • Ensure you clicked Refresh after changing inputs.

    Charts say no statistical data found

    • Confirm the filters include a period with activity.
    • Check if the selected template has recorded results.

    CSV export missing rows

    • The export follows the current filters; widen the date range.
    • Ensure you are in the correct tab for the data you expect.
