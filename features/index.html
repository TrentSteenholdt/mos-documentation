    Features Overview

    The Match Official System provides a complete set of tools to support match officials, administrators, and associations. It covers assessments, training, communication, reporting, and professional development.

    This page gives a clear, user focused overview of what the system can do and how features vary by role.

    Core Features for All Users

    Personal Dashboard

    Your dashboard is the first page you see after signing in. It gives you a personalised snapshot of your activity and anything that needs attention.

    Includes:

    • Welcome message
    • Recent news and announcements
    • Quick action shortcuts
    • Summary charts
    • Activity timeline

    Learn more: Home

    Profile Management

    Manage your personal details and account settings in one place.

    Includes:

    • Update personal information and contact details
    • Change password and security questions
    • Update email address
    • Upload or replace profile picture
    • Set notification preferences
    • View recent account activity

    Learn more: Profile Management

    Sign In and Account Access

    Secure access to the system and tools to recover your account if needed.

    Includes:

    • Invitation based registration
    • Secure sign in
    • Password recovery
    • Session management
    • Account security controls

    Learn more: Sign In
    Register

    News and Announcements

    Stay informed with updates published by your association and subgroups.

    Includes:

    • Read news articles
    • Download attachments
    • View announcements
    • Receive optional email notifications

    Learn more: News

    Assessments and Reviews

    Assessments and Reviews: Manage Your Development

    Create, submit, and track assessments and development activity over time.

    Includes:

    • Diary entries
    • Development plans
    • Assessment submission
    • Reviewer feedback
    • Assessment status tracking
    • Export assessment data

    Learn more: Assessments and Reviews

    Exams

    Complete online exams and review your results.

    Includes:

    • Multiple question types
    • Scoring and feedback
    • Pass marks and results
    • Exam retakes when permitted
    • Exam history tracking

    Learn more: Exams

    Library and Resources

    Folders

    Upload, organise, and share documents using the built in file manager.

    Includes:

    • Upload files and create folders
    • Rename and organise content
    • Move files between folders
    • Generate secure share links
    • Download files and folders

    Access Level

    Folder access depends on your role:

    • User Can upload to their personal folder (if enabled in group settings)
    • Viewer Can view subgroup folders they are assigned to (read-only)
    • Reviewer Can view subgroup folders they are assigned to (read-only)
    • Files Manager Can upload, delete, and organise files in group and subgroup folders
    • Owner Full administrative control of all folders

    Learn more: Folders

    Media

    Access files and media attached throughout the system.

    Includes:

    • Assessment evidence
    • News attachments
    • Profile images
    • Media downloads
    • Secure share links
    • Administrative clean up of unused images

    Learn more: Media

    File Downloads

    Download files from anywhere in the system.

    Includes:

    • Assessment evidence and exports
    • News attachments
    • Library files
    • Reports and data exports
    • Large file handling

    Learn more: File Downloads

    Reporting and Analytics

    Personal Reports

    View reports related to your own activity and performance.

    Includes:

    • Activity overview
    • Assessment history
    • Template performance
    • Exam results
    • CSV export

    Learn more: Reports

    Administrator Features

    Access Level

    The following features are only available to users with specific roles such as Viewer, Reviewer, Trainer, Communications, Files Manager, or Owner.

    User Management

    Add and manage users within your association.

    Access Level

    Owner access required.

    Includes:

    • Add and edit users
    • Reset passwords
    • Delete users
    • Export user lists

    Learn more: User Management

    Group Membership

    Organise members into subgroups and assign permissions.

    Access Level

    Owner access required.

    Includes:

    • Add and remove members
    • Assign permission levels
    • Create and manage subgroups
    • Bulk membership updates

    Learn more: Membership and Roles

    Group Settings

    Configure how your organisation operates within the system.

    Access Level

    Owner access required.

    Includes:

    • Subgroup structure
    • Theme and branding
    • Calendar settings
    • Quick links
    • Assessment templates
    • Registration controls

    Learn more: Group Settings

    News Management

    Create and publish news articles.

    Access Level

    Communications or Owner access required.

    Includes:

    • Create and edit news articles
    • Target specific audiences
    • Add attachments
    • Send email notifications
    • Unpublish or delete articles

    Learn more: News Management

    Data Management

    Clean and standardise team and venue data across your group.

    Access Level

    Owner access required.

    Includes:

    • View all unique team and venue values
    • Search and filter data entries
    • Merge duplicate or similar values
    • Standardise naming conventions
    • Update all assessments with canonical values

    Learn more: Data Management

    Assessment Management

    Oversee assessments across your group.

    Access Level

    Viewer, Reviewer, or Owner access required. Actions vary by role.

    Includes:

    • View and search all assessments
    • Reassign assessments and reviewers
    • Update assessment status
    • Bulk actions

    Learn more: Assessment Management

    Exam Management

    Create and manage exams and questions.

    Access Level

    Trainer or Owner access required.

    Includes:

    • Create exams
    • Add and manage questions
    • Set scoring and pass marks
    • Assign exams to users
    • Monitor attempts and results
    • Manual grading
    • Export results

    Learn more: Exam Management

    Group Reporting

    Generate reports across your organisation.

    Access Level

    Reviewer or Owner access required.

    Includes:

    • Login activity analysis
    • Assessment and performance metrics
    • Exam analytics
    • Observer engagement
    • Subgroup filtering
    • CSV and raw data export

    Learn more: Group Reporting

    Feature Enablement

    Feature availability can vary

    Some features are controlled by your association administrator and may not be enabled for all groups.

    Examples include:

    • User registration
    • Personal folders
    • Email notifications
    • Subgroup targeting

    Contact your administrator to confirm which features are enabled for your group.
