    Levels of Access

    Match Official System uses a flexible role-based permission system. Users are assigned an access level (User or Owner) plus any combination of specific roles. This allows fine-grained control over what each person can do.

    Overview

    Access is controlled through two components:

    1. Access Level: Either User (standard member) or Owner (full administrator)
    2. Roles: Specific permissions that can be combined as needed

    A person can have multiple roles at once. For example, someone could be a Reviewer and a Trainer, giving them permissions from both roles.

    Access Levels

    User User

    Description: Standard member access with personal workspace

    Permissions:

    • Create diary and plan entries
    • Search users in their group(s)
    • Sit exams
    • Access personal user folder (if enabled)
    • Upload files to personal user folder

    Owner Owner

    Description: Full administrative control of the group

    Permissions:

    • All User permissions
    • Create, edit, and delete user accounts
    • Modify group settings
    • Modify assessment templates
    • Manage exam questions and answers
    • Manage news and announcements
    • Full access to all folders and files
    • Assign roles to other members

    Roles

    Roles grant specific permissions and can be combined. A user can have multiple roles assigned.

    Viewer Viewer

    Permissions:

    • Create assessments (match day, media, season, general review) on other users
    • View assessment lists and history
    • View subgroup folders they are assigned to (read-only)

    Reviewer Reviewer

    Permissions:

    • Review diary and plan entries that are awaiting review
    • Provide feedback on shared assessments
    • Run reports on the group
    • View assessment lists (but cannot create assessments without Viewer role)
    • View subgroup folders they are assigned to (read-only)

    Trainer Trainer

    Permissions:

    • Access the Exam Management section
    • Create, edit, and manage exams
    • Export exam results
    • View exam completion activity

    Note

    The Trainer role is only available at the group level, not subgroup level.

    Communications Communications

    Permissions:

    • Access the News Management section
    • Create, edit, and publish news articles
    • Manage news distribution

    Note

    The Communications role is only available at the group level, not subgroup level.

    Files Manager Files Manager

    Permissions:

    • Create and delete folders
    • Upload and delete files in group and subgroup folders
    • Move and rename files and folders
    • Generate share links for files

    Common Role Combinations

    Here are some typical ways roles are combined:

    Person Type Access Level Roles
    Regular member User None
    Coach who reviews diaries User Reviewer
    Coach who creates assessments and reviews User Viewer + Reviewer
    Exam coordinator User Trainer
    News publisher User Communications
    File administrator User Files Manager
    Group administrator Owner (has all permissions)

    Understanding Folder Access

    Folder permissions depend on your access level and roles.

    Folder access for Users

    • Access only to personal folder: User/{username} (if enabled)
    • Upload and delete files only inside personal folder
    • Cannot access group or subgroup folders

    Folder access for Viewers

    • Can view subgroup folders they are assigned to (read-only)
    • Cannot upload, create, or delete files outside personal folder

    Folder access for Reviewers

    • Same as Viewer - can view subgroup folders they are assigned to (read-only)
    • Reviewer role is for reviewing diary and plan entries, not file management

    Folder access for Files Managers

    • Can create, rename, move, and delete folders and files
    • Can generate share links
    • Full file management within their assigned areas

    Folder access for Owners

    • Full folder control across the group and all subgroups
    • Can manage all files and folders
    • Can generate share links for any file

    Typical folder layout

    Home/
├── user/                   (personal user folders, if enabled)
│   ├── john.smith/
│   ├── jane.doe/
│   └── ...
├── subgroups/              (if enabled)
│   ├── Northern Division/
│   ├── Southern Division/
│   └── ...
├── Documents/
├── Photos/
└── Resources/

    Common Permission Questions

    Can I upload files?

    • User (no roles): Only to your personal folder
    • Viewer: Only to your personal folder
    • Reviewer: Only to your personal folder (Reviewer is for assessments, not files)
    • Files Manager: To all assigned folders
    • Owner: To all group and subgroup folders

    Can I delete files?

    • User (no roles): Only files you created in your personal folder
    • Viewer: No
    • Reviewer: No
    • Files Manager: Yes, in assigned areas
    • Owner: Yes, any file in any folder

    Can I review assessments?

    • User (no roles): No
    • Viewer: No (can create assessments, not review them)
    • Reviewer: Yes, for assessments awaiting review
    • Owner: Yes, all assessments

    Can I create exams?

    • User (no roles): No (can only sit exams)
    • Viewer: No
    • Reviewer: No
    • Trainer: Yes
    • Owner: Yes

    Can I publish news?

    • User (no roles): No
    • Viewer: No
    • Reviewer: No
    • Communications: Yes
    • Owner: Yes

    Can I run reports?

    • User (no roles): No
    • Viewer: No
    • Reviewer: Yes
    • Owner: Yes

    Can I manage users?

    • User (any roles): No
    • Owner: Yes, full user management

    Requesting Additional Roles

    If you need additional permissions:

    1. Contact your group administrator (Owner)
    2. Explain what tasks you need to perform
    3. They can assign the appropriate roles in Membership and Roles
