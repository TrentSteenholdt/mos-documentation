    Data Management: Clean and Normalise Field Values

    Access Level

    Owner access required

    Overview

    Data Management helps you clean and standardise data entries in your group. You can identify variations of the same team name or venue and merge them into a single canonical value.

    This ensures consistent data across assessments and makes reporting more accurate.

    How to Access

    1. From the main menu, click Admin
    2. Click Data Management

    What This Page Does

    Data Management shows you all unique values entered for teams and venues across all assessments in your group. You can search, select similar values, and merge them into one standardised entry.

    When you merge values, all existing assessments that used the old values will be updated to use the new canonical value.

    Page Sections

    Field Type Selection

    Two buttons let you choose which type of data to manage:

    • Teams - Shows all team names from Team A and Team B fields
    • Venue - Shows all venue names entered across assessments

    Click a button to view and manage that field type.

    Filter and Target Value

    Filter
    Type keywords to narrow down the list of values shown in the table below.

    Target Value
    Enter the canonical name you want to use after merging. All selected values will be changed to this name.

    Data Table

    The table shows:

    • Select checkbox - Check boxes to select values for merging
    • Value - The current name as it appears in assessments
    • Count - How many assessments use this value

    Values are listed alphabetically.

    How to Merge Values

    Merging combines multiple variations into one standardised name.

    Example Use Case

    Your assessments might have:

    • "Melbourne Tigers"
    • "Melbourne tigers"
    • "Tigers"
    • "Melb Tigers"

    You can merge all of these into "Melbourne Tigers".

    Steps to Merge

    1. Click Teams or Venue to select the field type
    2. Optional: Use the filter to search for similar values
    3. Check the boxes next to all values you want to merge
    4. Enter the canonical name in the Target Value field
    5. Click Merge Selected
    6. Review the confirmation modal showing:
      • All values being changed
      • The target value they will become
    7. Click Yes, Merge Values to confirm

    Result: All selected values are replaced with the target value across all assessments.

    Merge Confirmation Modal

    Before merging, a modal appears showing:

    Source Values
    All values you selected for merging

    Target Value
    The canonical name all values will become

    Warning Message
    A reminder that this action is irreversible

    You must confirm the merge by clicking Yes, Merge Values.

    Tips

    Check Before Merging
    Always review the list of source values and target value carefully before confirming.

    Start Small
    Try merging a few similar values first before tackling larger cleanups.

    Use Consistent Naming
    Establish naming conventions for teams and venues to reduce future variations.

    Refresh After Changes
    Click the Refresh button after merging to see the updated data table.

    Field Requirements

    Target Value, required
    You must enter a target value before the Merge Selected button becomes active.

    At Least Two Values Selected
    You must select at least two values to merge.

    Common Tasks

    Clean Up Team Name Variations

    1. Click Teams
    2. Use the filter to search for a team name
    3. Select all variations of the same team
    4. Enter the correct team name in Target Value
    5. Click Merge Selected
    6. Confirm the merge

    Standardise Venue Names

    1. Click Venue
    2. Scroll through the list or use the filter
    3. Select duplicate or similar venue names
    4. Enter the standardised venue name in Target Value
    5. Click Merge Selected
    6. Confirm the merge

    Search for Specific Values

    1. Select the field type
    2. Type keywords in the Filter field
    3. The table updates to show matching values only
    4. Click the X button to clear the filter

    Troubleshooting

    Merge Selected Button is Disabled

    Symptoms
    The button appears greyed out and cannot be clicked

    Solution

    1. Ensure you have selected at least two checkboxes in the table
    2. Ensure you have entered a value in the Target Value field
    3. Both conditions must be met to activate the button

    Changes Not Appearing

    Symptoms
    After merging, the old values still appear in the table

    Solution

    1. Click the Refresh button to reload the data
    2. The merged values should now show the updated canonical name

    No Values Appear in the Table

    Symptoms
    The table is empty after selecting a field type

    Common Causes

    • No assessments exist in your group yet
    • All existing assessments have empty values for this field
    • Data is still loading

    Solution

    1. Wait a moment for data to load
    2. Click Refresh to reload the data
    3. If still empty, create some assessments with team or venue data first

    Important Warnings

    Irreversible Action

    Merging values is permanent and cannot be undone. All assessments using the old values will be updated immediately. Always double-check before confirming a merge.

    Group Wide Impact

    Changes affect all assessments in your group, including historical records. Consider the impact on existing reports and statistics.
