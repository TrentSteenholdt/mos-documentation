    System Requirements

    Recommended Browsers

    Google Chrome (recommended)

    • Current version
    • Best overall performance and compatibility
    • Required for optimal features

    Microsoft Edge (recommended)

    • Current version
    • Full feature support
    • Good performance

    JavaScript Required

    JavaScript must be enabled in your browser to use the Match Official System. The system relies heavily on JavaScript for interactive features including:

    • Assessments and form submissions
    • Real-time data updates
    • File uploads and downloads
    • Dynamic content loading
    • Auto-save functionality
    • Media browsing and viewing
    • Rich text editing

    If you see a blank page or error messages: Check that JavaScript is enabled in your browser settings.

    Browser Compatibility

    While other browsers may work, Chrome and Edge provide the best experience. If you experience issues with another browser, try one of the recommended options above.

    Supported JavaScript versions: ES6+

    Hardware Requirements

    • Processor: Any modern processor (Intel/AMD)
    • RAM: Minimum 1 GB (4 GB recommended)
    • Storage: Minimal; files are stored in the cloud
    • Internet Connection: Broadband recommended (minimum 1 Mbps for file uploads)

    Mobile Devices

    The system is optimized for desktop and tablet use. Mobile phone access is supported but may have limited functionality depending on screen size.

    For best mobile experience:

    • Use a tablet (iPad, Android tablet) when possible
    • Use landscape orientation on some pages.
    • Consider switching to a desktop browser for complex assessments

    Network Requirements

    • Internet connection: Required (broadband recommended)
    • Firewall: May need to whitelist matchofficialsystem.com
    • Proxy servers: May require configuration in some corporate networks
    • Cookie support: Must be enabled in your browser

    Accessibility

    The system supports:

    • Screen reader compatible pages
    • Keyboard navigation
    • High contrast modes
    • Font size adjustment (browser zoom)
    • Mobile and tablet layouts

    For accessibility assistance: Contact your administrator or see FAQ

    Troubleshooting

    Blank page or won't load

    1. Clear browser cache and cookies
    2. Disable browser extensions temporarily
    3. Try a different browser
    4. Check your internet connection
    5. Contact your administrator if the problem persists

    Slow performance

    1. Close other browser tabs and applications
    2. Clear browser cache
    3. Try upgrading your internet connection
    4. Use a wired connection instead of WiFi if possible

    Features not working

    1. Ensure JavaScript is enabled
    2. Update your browser to the latest version
    3. Disable browser extensions temporarily
    4. Try in a private/incognito browsing window
    5. Try a different browser

    Large file uploads failing

    1. Check your internet connection
    2. Use a wired connection instead of WiFi
    3. Reduce file size if possible
    4. Try uploading during off-peak hours
    5. Contact your administrator for large file options

    Data Storage and Privacy

    • All data is stored in secure cloud servers in Microsoft Azure (Australia Southeast region)
    • Requires internet connection to access
    • Files stored in Azure Blob Storage
    • Database hosted on SQL Server
    • Data residency in Australia Southeast ensures compliance with local data storage requirements
    • No data is stored locally on your computer
